# Скилл: Xatab Parser (xatab_torent.py)

## Описание
Парсер для сайта byxatab.com - поиск игр, получение описания, скриншотов и ссылок на торренты.

## Доступные функции

### 1. xatab_search
Поиск игр по названию.

**Аргументы:**
- `query` (str) - название игры для поиска

**Пример использования:**
```python
results = xatab_search("assassin creed")
# Возвращает список игр с названиями, URL и картинками
```

### 2. xatab_game_details
Получить полную информацию об игре.

**Аргументы:**
- `game_url` (str) - URL страницы игры

**Пример использования:**
```python
game = xatab_game_details("https://byxatab.com/games/rpg/hades-ii/2-1-0-3310")
# game['title'] - название
# game['description'] - описание
# game['poster'] - постер/картинка
# game['screenshots'] - скриншоты
# game['download_url'] - ссылка на торрент
# game['tech_specs'] - системные требования
```

### 3. xatab_new_games
Получить список новых игр.

**Аргументы:**
- `page` (int, опционально) - номер страницы

## Примеры работы

### Поиск игры
Агент получает задачу: "Найди игру Hogwarts Legacy"
1. Вызвать `xatab_search("hogwarts legacy")`
2. Получить список найденных игр
3. Показать пользователю результаты

### Получить ссылку на торрент
Агент получает задачу: "Дай ссылку на торрент игры Hades II"
1. Вызвать `xatab_search("hades")` для поиска игры
2. Найти URL игры из результатов
3. Вызвать `xatab_game_details(URL)`
4. Извлечь `download_url` из результатов
5. Вернуть ссылку пользователю

## Важные Notes
- Некоторые игры с офлайн-активацией могут не иметь ссылки на торрент (доступна только покупка)
- Функции возвращают словари с полем `success` для проверки успешности
- При ошибке возвращается `{"success": False, "error": "текст ошибки"}`

## Интеграция с агентом
Агент должен автоматически использовать эти функции когда пользователь:
- Ищет игры для скачивания
- Просит найти торрент
- Запрашивает информацию об игре
- Хочет узнать системные требования
